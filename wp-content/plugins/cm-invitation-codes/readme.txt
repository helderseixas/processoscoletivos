**Add AJAX-based invitation popup login and registration forms to improve your website user login and registration experience and control who can register.**

Need more control over the login and registration of your WordPress site? The Registration and Invitation Codes Plugin is the best companion to any WordPress site. 

[vimeo https://vimeo.com/157189342]

The plugin supports creating a popup login and registration box, ajax registration, sign up code or login while allowing users to stay on the same page after login. Easily create or add invitation codes to the WordPress registration process. 

The pro version of the plugin supports email verification by all registered users, restricting WordPress registrations to users with invitation codes, sending fully customizable emails, auto remove users with unregistered emails, customize login labels, reCaptcha, adding additional registration forms and support a unique by role registration form.

== Installation ==

1. Upload plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Add a shortcode [cmreg-login]Text[/cmreg-login] or css class cmreg-login-click to a menu
