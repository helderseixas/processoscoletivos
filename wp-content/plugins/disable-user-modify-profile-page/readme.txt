Disable any user(except Admins), to access(modify) their profile page. 

== Description ==

 Under "All Users", individually per user, you can disable the profile page access.

== Installation ==
 
 
1) Unpack
2) upload to Wordpress plugins folder
3) Activate from Admin menu
